快速幂求解

来源：acwing 89

题目内容

求a的b次方对p取模的值。

输入格式

三个整数a，b，p，在同一行用空格隔开。

输出格式

输出一个整数，表示a ^ b mod p 的值。

数据范围

0 ≤ a, b, p ≤ 109

输入样例：

3 2 7

输出样例：

2
